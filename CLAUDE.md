# Post-task checklist

After completing any code changes:
1. Increment the version number in `src/lib/version.ts` (e.g. 1.052 → 1.053)
2. Commit all changes with a descriptive message that includes the new version (e.g. "Fix foo bar (v1.053)")
3. Push to `origin/main`
4. Create a backup to `/Users/doriteich/GamesSignup-backup/` (exclude `node_modules`, `.next`, `.git`)
